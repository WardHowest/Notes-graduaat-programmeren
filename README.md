## ✨ Formatting Guidelines
To keep everything organized I would like to stick to the formaating I made.
### SQL Formatting
- **Capitalization:** Start every sentence with a capital letter. Do not use ending punctuation.
- **SQL Clauses:** Always use **UPPERCASE** for clauses (e.g., `SELECT`, `FROM`, `WHERE`).
- **Variables/Placeholders:** Use `[type]` format. These are always lowercase (e.g., `[string]`).
- **Alignment:** To keep tables properly formated, use the [Markdown All in One](vscode:extension/yzhang.markdown-all-in-one) extension. Use the shortcut `Shift + Alt + F` to auto-format tables.

**Example:** `SELECT [column_name] FROM [table_name] WHERE [condition]`
